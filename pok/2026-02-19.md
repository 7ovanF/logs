to avoid an extra line used to put 0 into a register, 0 is automatically put into the reigster.

### Immediates
in MIPS, the argument either has to be a register OR an immediate/constant.

```
addi $s0, $s0, 4
     rt | rs | immediate
```

`addi` is a ddifferent instruction from `add` (it's an actual insruction, not a pseudo.ceek di green sheetnya)

This is an example of "make the common case fast".

> + immediates can be hex, bin, dec...
