# IP Address
[howstuffworks.com](https://computer.howstuffworks.com/web-server5.htm)

IP: Internet Protocol

TL;DR:
- Definition: unique address for a machine. Both for YOU to access a server and for a server to know where to send back data.
- Form: typically 32-bit numbers, in the form of four octets (xxx.xxx.xxx.xxx) where each xxx is an octet <- this is an IPv4 address.
- You only need an IP address to access a machine, but a human *probably* needs a better way to remember them: Domain names. Form: www.website.com.
- You can see the IP address of a machine with the domain name with `nslookup`
