# Domain Name

Definition: human-readable string that acts as an address used to identify a server, each of which are mapped to the server's IP address through the [Domain Name System (DNS)](4-DNS).

Example: google.com
Parts:
- com is a top-level domain name (TLD).
- google is a second-level domain name (SLD).
- on the other hand, www is a [hostname](5-Hostname.md).
See more: [Domain name space (+ TLD, SLD and lower)](https://en.wikipedia.org/wiki/Domain_name#Domain_name_space)

You can delve into the details of a certain domain using its domain name by using `whois (domain)` which will consult the RFC 3912 database.

Important read: [Domain Name System (DNS)](4-DNS.md).
