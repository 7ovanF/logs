# Domain Name System (DNS)

Definition: **"a hierarchical and distributed name service that provides a naming system for computers, services, and other resources on the Internet or other Internet Protocol (IP) networks."** Src: [Wikipedia:Domain Name System](https://en.wikipedia.org/wiki/Domain_Name_System)

Enforced by DNS Servers as medias to resolve domain names into IP addresses. There are categories of DNS Servers, each responsible for a certain domain level.
