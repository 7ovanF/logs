# Make

### Making vs Compiling
Compile: directly from a source code file into its compiled form, as a usual compiler does.
Make: It will follow directions from a **makefile**. The directions in a makefile usually aim to turn source codes into a cohesive structure, even into an installed app. Directions include calling compilers themselves (make programs DON'T compile in the traditional sense) along with doing some dependency management. 

---
A popular `make` program is GNU Make.
