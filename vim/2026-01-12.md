- :![command] does an external command (any, such as ls)
- :w [filename] writes to that file even if it doesnt exist
  + you can write into a different folder as long as it exists
  + you can write only from a certain range

Why so many basics here
- R (capital) [replace multiple] enters REPLACE MODE (kinda like accidentally pressing insert)
- y and p - yank
- Searching options (rules, use :set)
  -> ic (ignore case), \c for only one time, set noic to revert
  -> hls (highlight search) highlight
  -> is (includes(inclusive?) search)
